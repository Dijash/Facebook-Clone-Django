# Facebook Clone (Django)

A modern Facebook-inspired social media web application built using **Django**. This project replicates the core functionalities of Facebook, allowing users to create accounts, manage profiles, share posts, and interact with other users through a clean and responsive interface.

---

## 📌 Features

* 🔐 User Authentication (Login, Register & Logout)
* 👤 User Profile Management
* 📝 Create, Edit and Delete Posts
* 📰 News Feed Display
* ❤️ Like Posts
* 💬 Comment on Posts
* 🖼️ Profile Pictures
* 🔍 Search Functionality
* 📱 Responsive User Interface
* 🎨 Facebook-inspired Design

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Dijash/Facebook-Clone-Django.git
```

### 2. Navigate to the Project

```bash
cd Facebook-Clone-Django
```

### 3. Create a Virtual Environment

#### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

---

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 5. Apply Database Migrations

```bash
python manage.py migrate
```

---

### 6. Run the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
